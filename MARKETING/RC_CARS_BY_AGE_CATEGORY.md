## 📊 PORTFOLIO STRATEGY (BCH 4-tier shelf)

Map directly to store shelves:

```
┌──────────────────────────────────────────────────────────────────┐
│  SHELF 1: GIFT ENTRY (Pre-school 3-6 yrs)                        │
│  Price: ₹500-1,500 | Source: Sadar Bazaar wholesale              │
│  Brands: Centy, Toyzone, Mega Deals, B Shah                      │
│  Margin: 50-70% (cheap product, high markup)                     │
├──────────────────────────────────────────────────────────────────┤
│  SHELF 2: KIDS MAIN (7-12 yrs)                                   │
│  Price: ₹2,000-4,500 | Source: LOT direct + Bhawani Chennai      │
│  Brands: LOT Flare 2.0/Ghost/Shadow, Toyshine, Webby, Toofun     │
│  Margin: 35-50% (where most volume happens)                      │
├──────────────────────────────────────────────────────────────────┤
│  SHELF 3: TEEN BRIDGE (13-17 yrs)                                │
│  Price: ₹5,000-15,000 | Source: MJX/Tygatec direct + Daddy Drones│
│  Brands: MJX Hyper Go, LOT Apex/Doughty/Gazer, Tygatec Supersonic│
│  Margin: 25-40% (bridge to hobby)                                │
├──────────────────────────────────────────────────────────────────┤
│  SHELF 4: HOBBY ADULT (18+)                                      │
│  Price: ₹15,000-2L+ | Source: Hemani Exim Mumbai                 │
│  Brands: Kyosho, ARRMA, Traxxas, FMS, Tamiya                     │
│  Margin: 20-35% (status / club / aspirational)                   │

```

**Customer journey:** Birthday gift Cat1 → Birthday Cat2 → Asks for upgrade Cat3 → Hobby commit Cat4 over 5-8 years per family. BCH wins by being the only Bangalore retailer that has ALL FOUR shelves stocked simultaneously.
