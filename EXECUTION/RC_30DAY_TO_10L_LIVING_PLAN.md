### TASK 2 — Build Multiple RC Tracks (for content production)
**Owner:** Syed + ops team | **Deadline:** Day 7 (2026-06-01) | **Status:** 🟡 NOT STARTED | **Budget:** ₹20,000-35,000 (DIY) or ₹50,000-1L (premium)

**Why multiple tracks:**
- Different track types = different content angles = avoids repetition fatigue in reels
- Tracks = built-in props that customers can DRIVE on when they visit BCH = experience-driven retail
- Track sessions = paid revenue line (₹100-250/30min like RACEiN PARC)

**Three tracks to build, in priority order:**

**Track A — Drift Track (DIY, indoor)**
- Surface: smooth ceramic tile floor or polished concrete + masking-tape track outline
- Dimensions: ~3m × 2m
- Obstacles: 4 cardboard "drift markers" (cones) + 2 hairpin turns
- Cost: ₹500 (tape + cones)
- Time: 1 day
- **Use case:** Mini RC drift content (the @manavladha 17.99M-view format)

**Track B — Crawler Track (outdoor, BCH parking)**
- Surface: real ground / rocks / bricks / wood planks creating uneven terrain
- Dimensions: ~4m × 3m
- Obstacles: stacked bricks, water trough (for crawler-in-water content), a wooden ramp
- Cost: ₹3,000-5,000 (lumber + bricks)
- Time: 1 day
- **Use case:** TRX-4M Defender / LOT Knox / hobby crawler content (proven viral — Tank 300 reel 4.34M views)

**Track C — Speed/Drag Track (BCH parking strip)**
- Surface: long paved area, ~10-15m straight
- Markers: start line + finish line tape + spray paint speedometer markings
- Cost: ₹200 (tape + spray)
- Time: 30 minutes
- **Use case:** "0-60 in X seconds" / "Top speed test" content (MJX Hyper Go 10208 top speed reel got 4.8M views)

**Action items:**
- [ ] Day 5: Order materials from Amazon + local hardware shop
- [ ] Day 6: Build Track A (drift) — film 5 reels same day
- [ ] Day 7: Build Track B (crawler) — film 5 reels including "in water"
- [ ] Day 7: Mark Track C in BCH parking — film top-speed test
- [ ] Day 8+: Rotate tracks weekly for content variety

**Future upgrade (Month 2):** Permanent fiberglass drift track like RACEiN PARC (₹2-5L investment if revenue justifies).
